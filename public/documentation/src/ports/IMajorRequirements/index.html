IMajorRequirements path helpers

Two functions that translate raw folder names from the requirement data source into UI-ready display strings. The actual file fetching stays in the loaders — this port owns only the institution-specific name-parsing logic.

Port key
"majorRequirements"
Consumed by
majorLoader.js, minorLoader.js (passed as parameter) → GradPanel major/minor selectors
Vite constraint
import.meta.glob() requires string literals — glob patterns stay in the loaders, not here
Override when
Your folder naming convention differs from snake_case or has location suffixes

Contract

FunctionSignaturePurpose
fmtLabel(folderName: string) → stringConvert a raw data source folder name to a human-readable major/minor label shown in the selector dropdown.
fmtLocation(folderName: string) → stringExtract a location tag from a folder name. Return empty string if the institution has no location concept for majors.

Northeastern reference implementation

NU's requirement data (from external/graduatenu) uses folder names like "computer_science_(boston)" that embed a campus location in parentheses. The adapter strips the location tag for the label and extracts it separately.

Input folder namefmtLabel resultfmtLocation result
computer_science_(boston)Computer ScienceBoston
electrical_engineering_(oakland)Electrical EngineeringOakland
mathematicsMathematics(empty string)
bs-csBS CS(empty string)

Short all-lowercase words (≤3 chars) like "bs" and "cs" are uppercased as acronyms. Underscores and hyphens become spaces. Parenthetical location tags are stripped before title-casing.

Generic default

Same title-case logic as NU but without location extraction. fmtLocation always returns "".

InputfmtLabel (generic)fmtLabel (NU)fmtLocation (generic)fmtLocation (NU)
computer_science_(boston)Computer Science (Boston)Computer Science""Boston
mathematicsMathematicsMathematics""""
✦
The generic default works fine for most naming conventions. Only override if your folder names have special characters, non-English text, or embedded metadata (like NU's campus tags) that need stripping.

Why not in the loaders?

import.meta.glob() is a Vite build-time construct that requires string literals at the call site — it cannot be abstracted into a function call. This means majorLoader.js and minorLoader.js must contain the glob patterns directly. But the string-manipulation functions (fmtLabel, fmtLocation) are institution-specific logic that belongs in the adapter. This port is the seam between the two.
