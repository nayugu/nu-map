Update Workflow

      

        

          1
          
Fetch fresh course list from the SearchNEU API:
npm run data:fetch → review diff → npm run data:fetch:write

        

        

          2
          
Optionally scrape the catalog for richer descriptions:
npm run data:scrape:write

        

        

          3
          
Re-apply manual patches:
npm run data:patch:write

        

        

          4
          
Build and verify:
npm run build

        

        

          5
          
Open a pull request — GitHub Actions validates and posts a diff summary.

        Use the Patch Creator tab to generate YAML,
        then place it in that folder and open a PR.
        Naming convention: YYYY-MM-DD_short-description.yaml

Quick Actions

      
Run these commands in the repo root. Build afterwards to apply changes.

      

        

          

            
Annual course refresh

            npm run data:fetch:write && npm run data:patch:write && npm run build
          

          
        

        

          

            
Scrape from catalog.northeastern.edu

            npm run data:scrape:write && npm run build
          

          
        

        

          

            
Validate all patches

            npm run data:validate
